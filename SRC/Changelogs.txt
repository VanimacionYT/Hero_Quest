[Este proyecto está desarrollado en el lenguaje "Python" usando el IDE de "Jupyter Notebook"]

Hero Quest es un juego desarrollado a partir de una tarea de mi academia de python el cual he decidido mantener actualizano, añadiendo todas las funcionalidades útiles que vaya aprendiendo.

Este juego no es mas que un mero proyecto académico.

En este documento de texto "Changelogs.txt" redactaré los cambios que se reporten en el juego.

__[Changelog 1.0]__
- El juego es creado con las características base. El metodo de juego es simple, se inicia la partida, se escojen los parámetros de cada personaje y, a partir de un método de dado virtual, se crean turnos por los cuales se ataca o defeiende cada personaje hasta que la vida de uno de ellos llegue a 0 y se acabe la partida.
